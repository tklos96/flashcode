NAME

Driver_computeDt

DESCRIPTION

Determine the stability-limited time step. This timestep is determined using information from the included physics modules - many different timestep limiters are polled.

The global driver might use a different (hopefully smaller) time step, to match a file write time (tplot or trstr) or if the simulation end time has been reached; such possibilities are not considered here.

NOTES

The Driver unit uses a few unit scope variables that are accessible to all routines within the unit, but not to the routines outside the unit. These variables begin with "dr_" like, dr_globalMe or dr_dt, dr_beginStep, and are stored in fortran module Driver_data (in file Driver_data.F90. The other variables are local to the specific routine and do not have the prefix "dr_"

The calls to units currently not included in the code are commented out.
