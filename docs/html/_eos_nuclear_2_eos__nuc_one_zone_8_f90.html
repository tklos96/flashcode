NAME

Eos_nucOneZone

DESCRIPTION

Routine for cranking the EOS on a single zone, in any old mode you like. Calls kernel routine nuc_eos_full. Can also be used to recover values from the table that are not provided by nuc_eos_short without doing a full EOS solve.
