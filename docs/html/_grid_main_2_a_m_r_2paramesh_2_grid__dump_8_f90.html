NAME Grid_dump

DESCRIPTION

Dumps the variables specified in "var" to a file. Can be done from anywhere in the code, and is useful for diagnostic purposes With paramesh this function doesn not work in parallel, but works only with a single block

EXAMPLE

num = 3 !dumping 3 variables var(1) = DENS_VAR var(2) = PRES_VAR var(3) = TEMP_VAR blockID = 1 ! local block number gcell = .false.

call Grid_dump(var, num, blockID, gcell)

will dump the interior cell values of density, pressure and temperature for local block number 1.

NOTES DENS_VAR, PRES_VAR, TEMP_VAR etc are #defined values in Flash.h indicating the index in the physical data array. The routine calling Grid_dump will need to include Flash.h

This routine works with Paramesh only under very special circumstances
