NAME gr_xyzToBlock

DESCRIPTION

This routine returns the identity of the block, on which the specified physical location falls. It also returns the ID of the processor on which the block is residing.

If the physical location lies outside the domain boundaries, then procID=NONEXISTENT will be returned; boundary periodicity is not considered.

NOTES

This implementation requires BITTREE.
