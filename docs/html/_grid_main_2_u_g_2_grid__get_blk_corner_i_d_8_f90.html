DESCRIPTION

Returns the global integer indices of the left most interior zone of the block and the stride of indices along each dimension. Together the cornerID and the stride make a unique identifier for each block on the grid.

Since Uniform Grid does not have different levels of refinement, the value of stride is always one and the cornerID can be calculated as me(axis)*blkLimits(axis), where me(axis) is the processor number along desired axis and blkLimits(axis) is the size of the blocks along the same axis.

CornerID counting starts at 1 and only the interior cells (no guardcells) are used to calculate the cornerID.

EXAMPLE

In a 1 dimensional UG case with 2 blocks and nxb=8 The cornerID for block 1 = 1 and the cornerID for block 2 = 9
