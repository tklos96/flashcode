NAME gr_hgMapBcType

DESCRIPTION This routine returns a boundary condition type (BC type) for a specific variable in a specific grid datastructure, based on the BC type that is stored in the structures of the Grid that keep track of block neighbors.

Normally, this routine just returns the bcTypeFromGrid as bcTypeToApply. However, this routines gives the HG solver the opportunity to apply different types of boundary conditions from what the Grid would normally use.
