NAME

gr_mpoleMom1Dspherical

DESCRIPTION

Prepares for evaluation of the moments in 1D spherical geometry. In this routine, all the necessary arrays are prepared to enable evaluation of the moments in radial bin order. Each of the moments are grouped together according to their radial bins. This will ensure optimum unit stride values when accessing the big moment arrays and makes threading trivial.
