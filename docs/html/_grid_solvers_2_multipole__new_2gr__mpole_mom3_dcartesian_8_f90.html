NAME

gr_mpoleMom3Dcartesian

DESCRIPTION

Prepares for evaluation of the moments in 3D cartesian geometry. In this routine, all the necessary arrays are prepared to enable evaluation of the moments in radial bin order. Each of the moments are grouped together according to their radial bins. This will ensure optimum unit stride values when accessing the big moment arrays and makes threading trivial.
