Parameters

pfft_outArrayarray that is output from pfft NOTE Users must make sure that Grid_pfftInput has been called correctly before calling this routine.

NAME

Grid_pfftMapFromOutput

DESCRIPTION

The reverse of Grid_pfftMapToInput: Places the data from the PPFT array pfft_outArray into the mesh at variable gridVar. Once again it uses the map determined by the routine Grid_pfftInit.
