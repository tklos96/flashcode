NAME gr_hgSolve

DESCRIPTION This is the main Poisson solve routine for the Huang & Greengard (2000, SIAM J. Sci. Comput., 21, 1551) algorithm. This routine defines the multigrid cycle as expressed in the article.

coarse ^ o-—>o o-—>o ^ s \ ^ s \ c r| o o |r o o o e| l \ |e l \ r s| v o |s v o r t| e \ |t e \ e r| o |r o c i| \ |i \ t c| o |c o t| \ |t \ src | o--—>o gr_iSls o---—>...until |r| < rtol fine |______________take residual____________take new residual__>

EXAMPLE

gr_hgSolve(idens, gpot, hgwk1, hgwk2, gr_hgPoissonSolveBlock, (/(MG_BND_DIRICHLET,i=1,6)/), newton)

This call to this equation takes the density variable and puts a 0-boundary-valued solution to poisson's equation in gpot, using hgwk1 and hgwk2 as work variables (for the residual and correction)

USED BY

Primarily Grid_solvePoisson.
