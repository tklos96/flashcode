NAME gr_hgSolveLevel

DESCRIPTION

Block-per-block solve Poisson on all the blocks at level using boundary conditions on the exterior faces. The second phase of this routine does two Jacobi sweeps on the first two interior cells for each block in order to dampen inter-block discontinuities.

NOTES

The Jacobi sweeps SHOULD be replaced with something for better convergence.
