NAME pt_mapStringParamToInt

DESCRIPTION

This routine is used for finding the integer value associated with some string based runtime particles parameters that occur in groups and are therefore enumerated. For example for runtimte parameters such as "particles_attributes_1", "particles_attributes_2" etc, this routine takes paramString as "particles_attributes_", ind provides the enumeration value, and the associated constant is returned in mappedInt
