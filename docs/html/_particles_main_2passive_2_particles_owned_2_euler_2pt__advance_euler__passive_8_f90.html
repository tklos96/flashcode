NAME

pt_advanceEuler_passive

DESCRIPTION

Time advancement routine for the particle module.

This version implements 1st order Euler integration in time. SCHEMATICALLY: x(t+1) = x(t) + dtNew * v(t)

In detail: x(t+dtNew) = x(t) + dtNew * v(x(t), t) v(t+dtNew) = v(x(t+dtNew),t+dtNew)

NOTES

No special handling is done for the first call - it is assumed that particle initialization fills in initial velocity components properly.
