NAME PhysicalConstants_init

DESCRIPTION

This subroutine initializes the Physical Constants databases for units and constants. Must be called in the Simulation_init

pc_unitsBase [default "CGS"] set the default system of units, either "CGS" or "MKS" NOTES

If you see a constant named "Newton", what is meant is probably not the SI unit of force which has dimensions LENGTH^1 * TIME^(-2) * MASS^1, but Newton's gravitational constant G.

Using the "physical constant" "pi" is deprecated, use PI from "constants.h" instead.
