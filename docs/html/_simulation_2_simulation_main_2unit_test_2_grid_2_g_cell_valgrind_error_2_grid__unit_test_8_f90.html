Parameters

NAME

Grid_unitTest

DESCRIPTION

This routine tests the internal operation of meshes Specifically it tests guardcell filling with the routine Grid_fillGuardCells. If all tests pass then a file usually named unitTest_000x is written with the line, "All tests conformed to expected results". If the tests failed, various error messages may be written to the file.
