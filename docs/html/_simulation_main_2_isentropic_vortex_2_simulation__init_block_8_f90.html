NAME Simulation_initBlock

DESCRIPTION Initialize fluid data for a specified block.

This version initializes conditions for the isentropic_vortex problem. Given a constant background state specified by density rho_o, pressure p_o, and velocity (u_o, v_o). Define the following perturbations: du = -y (vortex_strength/ (2 pi)) exp ((1 - r^2)/2) dv = x (vortex_strength/ (2 pi)) exp ((1 - r^2)/2) dT = - ((gamma-1)/2) (vortex_strength/ (2 pi))^2 exp (1 - r^2) where x and y are the distance from the vortex center, and r^2 = x^2 + y^2

Then the conserved variables at time=0 are rho = (T_o + dT)^(1/(gamma-1)) rho u = rho (u_o + du) rho v = rho (v_o + dv) rho e = (rho^gamma)/(gamma-1) + rho (u^2 + v^2)/2

NOTES Reference: Yee, Vinokur & Djomehri, J. Comp. Phys 162
