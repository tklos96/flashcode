NAME

Simulation_initParticleAttrib

DESCRIPTION

A stub for initializing additional particle attributes if necessary.

This interface is called during initialization after o particles positions, o particle velocities and other properties needed to advance particle trajectories, o mass properties for active particles, o and particle tags have all been initialized (or possibly read from a checkpoint, if restart is true).

This implementation initializes PosInit{X,Y,Z} particle properties to the corresponding Pos{X,Y,Z} values.
