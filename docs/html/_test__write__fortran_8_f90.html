NAME

Test_write_fortran

DESCRIPTION

This is a standalone program that tests I/O independently of FLASH

NOTES

We assume code is compiled with an option that promotes 4 bytes reals to 8 byte reals.

We want to write the internal region of dataArray to file. This is: dataArray(1, 2:3, 2:4, 1, 1) as NXB=2, NYB=3, NGUARD=1.

In Fortran the left most dimension in a multi-dimensional array is contiguous. So for an array(i,j), I like to draw the contiguous axis (i) on the column because Fortran is column major.
