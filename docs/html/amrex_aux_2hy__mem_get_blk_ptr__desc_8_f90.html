NAME hy_memGetBlkPtr

DESCRIPTION

Gets a pointer to a single block of allocated scratch data that conforms with the specified Grid data structure.

The block data may include zero, one, or several layers of guard cells, dependent on how hy_memAllocScratch was called for the gridDataStruct. If the optional argument "gridDataStruct" is not specified, it returns a block from cell centered data structure.

NOTES

hy_memGetBlkPtr is an accessor function that passes a pointer as an argument and requires an explicit interface.

If you call hy_memGetBlkPtr, you should also call hy_memReleaseBlkPtr when you are done using the pointer.
