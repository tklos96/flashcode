NAME

bn_azbar

DESCRIPTION

routine azbar computes composition variables from the mass fractions

given the mass fractions in xmass(i), return the molar abundances ymass(i), total number of moles per gram ytot1, the mean number of nucleons abar, mean nucleon charge zbar, mean nucleon charge squared z2bar, and the electron mole number bye.

NOTES the output variables are stored in data structure Burn_dataEOS
