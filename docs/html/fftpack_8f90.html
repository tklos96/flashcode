NAME

fftpack

DESCRIPTION

A collection of routines for computing Fast Fourier Transforms (FFTs).

NOTES

http://www.netlib.org/fftpack/:

"FFTPACK is a package of Fortran subprograms for the fast Fourier transform of periodic and other symmetric sequences. It includes complex, real, sine, cosine, and quarter-wave transforms. Reference P.N. Swarztrauber, Vectorizing the FFTs, in Parallel Computations (G. Rodrigue, ed.), Academic Press, 1982, pp. 51–83. Developer Paul N. Swarztrauber, National Center for Atmospheric Research, Boulder, CO"

This is a Fortran 90 translation of the fftpack package described above. Translation by John Burkardt. Code is freely available from http://people.scs.fsu.edu/~burkardt/f_src/fftpack/fftpack.f90.

This code has been slightly modified by storing the value of pi to a higher precision.
