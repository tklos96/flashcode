NAME

flash_unconvert_cc_hook

DESCRIPTION

Given some block of data in DATAINOUT, loop over all of the blocks and convert variables which are normally represented in PER_MASS form (e.g., velocity) back from corresponding conservative form (i.e., momentum) to their normal representation.

In other words, undo the effect of flash_convert_cc_hook.

NOTES

The variables that are converted are the named cell-centered solution variables marked to be of type PER_MASS explicitly in a Config file. Additionally, abundances and mass scalars are considered to be of type PER_MASS.

For proper functioning, DENS_VAR must not be marked as PER_MASS!
