NAME

gr_bhBuildTree

DESCRIPTION

Builds the tree for the Poisson solver. The first part builds trees in all LEAF blocks. The second part distributes masses and mass centres of all blocks to all CPUs (array gr_bhTreeParentTree). The third part calculates masses and mass centres of all parent blocks (also in the array gr_bhTreeParentTree).
