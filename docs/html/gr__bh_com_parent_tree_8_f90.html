NAME

gr_bhComParentTree

DESCRIPTION

Communicates the array gr_bhTreeParentTree which includes masses and mass centres of all blocks. Values for blocks at a given refinement level are distributed to all CPUs. If level = -1, values for all LEAF blocks are distributed.
