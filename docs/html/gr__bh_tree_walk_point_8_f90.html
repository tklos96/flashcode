NAME

gr_bhTreeWalkBlockPoint

DESCRIPTION

Executes tree walk for a point given by physical coordinates x, y and z. In this version it needs to know also the target cell (specified by block and point) in which the point x,y,z lies, because *NodeContrib routines write calculated information into it. This should be rewritten in a cleaner way at next bigger code restructuring.
