NAME gr_copyGuardcellRegionToFab

DESCRIPTION This routine is used to copy guardcell data from a special data structure filled by Grid_bcApplyToRegion into the AMReX FAB that is requesting data for guardcells that lie outside the domain.

This routine assumes that the fab, guardcells, and region data structures are all specified with respect to the same index space.
