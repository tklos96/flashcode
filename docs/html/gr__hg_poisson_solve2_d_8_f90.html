NAME gr_hgPoissonSolve2D

DESCRIPTION

Solve Poisson's equation on a 2D rectangular domain using Fourier, sine, or cosine transforms. The type of transform employed depends upon the requested boundary conditions. Transforms are performed using Takuya Ooura's FFT package (http://momonga.t.u-tokyo.ac.jp/~ooura/fft.html).
