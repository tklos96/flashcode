NAME gr_hgSetExtBoundary

DESCRIPTION

Set boundary values of the exterior guardcells of leaf blocks in the work array such that the given boundary value is satisfied. The handled cases are MG_BND_DIRICHLET, MG_BND_NEUMANN, MG_BND_GIVENVAL

Periodic boundaries are handled topologically and do not need a case here. Allegedly this routine also handles cylindrical and spherical coordinates.
