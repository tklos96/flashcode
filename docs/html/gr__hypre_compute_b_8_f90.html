NAME

gr_hypreComputeB

DESCRIPTION Computes the RHS of AX=B using a precomputed matrix M (stored in diff_A) such that B=MX. A MatVec product is performed to compute B and additional source terms are added if required. A lot of the arguments are not required by this routine but are never the less provided so that B can be computed outside of HYPRE (if needed).
