NAME

gr_hypreGetFaceBFcB

DESCRIPTION This routine helps to compute iFactorB*AREA on fine-coarse boundaries.

This FcB ("face-centered B") variant expects the diffusion coefficients to be given in the face-centered POINTER array facBptr.
