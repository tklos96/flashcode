NAME gr_initNewLevelCallback

DESCRIPTION This routine is a callback routine that is registered with the AMReX AMR core at initialization. AMReX calls this routine to create a new refinement level from scratch.

Specifically, for the given refinement level it (1) creates a multifab for each data type, (2) initializes these structures with the initial conditions via Simulation_initBlock, (3) fills all cell-centered guardcells, and (4) runs EoS on the interiors and GCs of all blocks to make the initial data thermodynamically consistent.

Note that all EoS runs are done in the mode specified by the eosModeInit runtime parameter.

This routine should only be invoked by AMReX.
