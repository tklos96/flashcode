NAME

gr_pfftFnArgEasyConstraint

DESCRIPTION

This is a very easy constraint to satisfy. We are only interested in generating a PFFT grid in which we use all available processors.

We should always be able to satisfy this contraint with grids of [1,1,totalProcs], and [1,totalProcs,1]. Failure to do so is an error!
