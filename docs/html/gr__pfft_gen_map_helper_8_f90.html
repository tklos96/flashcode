NAME

gr_pfftGenMapHelper

DESCRIPTION

This subroutine is a helper subroutine for gr_pfftGenMap. It calculate a metadata quantity named PFFT_NUMBLKS in the map structure. This describes the number of blocks moving to each processor. We also return maxSingleProcData, which is an integer variable used to size our send/recv buffer containing the actual block data.
