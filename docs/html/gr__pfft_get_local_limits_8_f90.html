NAME

gr_pfftGetLocalLimits

DESCRIPTION

Find out the local index limits of the transformed data in a processor. This imformation is necessary to calculate the derivatives. Here the situation is complicated by the fact that after transformation, the order of physical dimension is <JAXIS,IAXIS> in 2D and <KAXIS,IAXIS,JAXIS> in 3D. So the ordering of dimensions in the array is different from the physical dimension order. The two arguments passed to this function provide the mapping.

NOTES

Indices in pfft_localLimit are 0-based.
