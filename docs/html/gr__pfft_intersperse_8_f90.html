nx1the factor by which nx is split

NAME

gr_pfftIntersperse

DESCRIPTION

This is a helper routine for the forward distributed transpose. It only moves the data around, but doesn't change it. If we were to visualize inArray as having the dimensions (nx/nx1,ny,nx1), then the outArray is shaped as (nx,ny)

EXAMPLE

For nx=8, ny=4 and nx1=2, the array changes shape as follows

1,1 1,2 1,3 1,4 5,1 5,2 5,3 5,4 2,1 2,2 2,3 2,4 6,1 6,2 6,3 6,4 3,1 3,2 3,3 3,4 7,1 7,2 7,3 7,4 4,1 4,2 4,3 4,4 8,1 8,2 8,3 8,4

to

1,1 1,2 1,3 1,4 2,1 2,2 2,3 2,4 3,1 3,2 3,3 3,4 4,1 4,2 4,3 4,4 5,1 5,2 5,3 5,4 6,1 6,2 6,3 6,4 7,1 7,2 7,3 7,4 8,1 8,2 8,3 8,4

NOTES

Also see gr_pfftDisperse, the action of this routine is exact opposite of that of gr_pfftDisperse
