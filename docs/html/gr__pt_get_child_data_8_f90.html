NAME gr_ptGetChildData

DESCRIPTION

Helper routine for gr_ptFindNegh. Obtains the block and process ID of the neighbor LEAF blocks at relative position specified by guardCellID. These neighbor blocks are at a finer refinement level than our source block. Our 'real' neighbor is a PARENT which is identified by neghBlkID, neghProcID.
