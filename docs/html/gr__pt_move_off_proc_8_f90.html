NAME

gr_ptMoveOffProc

DESCRIPTION

This routine is used in moving the particles data when a particle moves off a block because of time integration. The routine examines a single face of the block at a time. Here every particle currently on the processor is examined to see if it has moved out of the block across the specific face and axis under consideration. If the face was along physical boundary, then it may either leave the domain, stay on the same block or be moved to gr_ptDestBuf, which holds particles to be passed to the next processor, depending on the boundary conditions. Once all particles have been examined, the contents of gr_ptDestBuf are sent to the neighbor along the face, and the contents of opposite face neighbors gr_ptDestBuf are received. The newly received particles are added to the particles data structure. Note that it is assumed that no particle can move across a full block in a single time step.
