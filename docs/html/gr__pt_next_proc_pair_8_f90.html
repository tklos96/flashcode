NAME gr_ptNextProcPair

DESCRIPTION

This routine finds the pair of processors with which MyPe must communicate in the next iteration of sieve motion. The routine is abstrated out from gr_ptMoveSieve to allow different algorithms for picking the next processor pair. For example in some situations the processor pair may remain fixed, as when sorting processors based on their tags among processors. In the default mode, the number the motion is back-and-forth with increasing intervals
