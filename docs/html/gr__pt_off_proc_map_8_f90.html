NAME gr_ptOffProcMap

DESCRIPTION

This routine copies the guard cell region into the one-dimensional send buffer. Data is stored as metadata + smeared guard cells. The smeared guard cells are stored in increasing memory order (rather than contiguous order) in the send buffer. The coordinates inside the guard cell region are converted into an appropriate 1D offset. Therefore, the end destination of the data, can be obtained from the position of the guard cell in the send buffer and the metadata containing the coordinates of the actual guard cell region.
