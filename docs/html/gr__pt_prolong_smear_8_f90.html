prolongedSectionAn array containing the interpolated data. (This is the data that needs to be accumulated on the finer mesh).

NAME

DESCRIPTION

This routine is a wrapper around the FLASH2 prolongation subroutine. It uses a single internal cell from the source block, along with its low / high neighbors in each dimension, to approximate the value that should exist in the cells that occupy the same spatial location as the single internal cell but on a finer mesh. This technique is known as prolongation (interpolation)
