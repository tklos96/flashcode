gr_ptVerifyBlock(real(IN) :: particles(:,:), integer(IN) :: propCount, integer(IN) :: localNumParticles, integer(IN) :: maxParticlesPerProc)

Parameters
particlesList of particles. It is a 2D real array, the first dimension represents particle's properties, and second dimension is index to particles.
propCountThe count of fields in the particles data structure
localNumParticlesThe current number of particles mapped to this processor.
maxParticlesPerProcThis is parameter determined at runtime, and is the maximum number of local particles that a simulation expects to have. All the arrays in the particles unit are allocated based on this number

NAME gr_ptVerifyBlock

DESCRIPTION

Used to check whether each particle is associated with:

  1. A valid block (i.e. not NONEXISTENT)
  2. The correct block (i.e. it exists within this block's bounding box).
