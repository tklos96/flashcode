NAME

hy_HLLC

DESCRIPTION

This routine computes high-order Godunov fluxes based on the left and right Riemann states.

The HLLC Riemann fan: 

     SL      SM=qStar    SR
      \        |        /
       \       |       /
        \  UL* | UR*  /
         \     |     /
          \    |    /
           \   |   /
    UL      \  |  /       UR
             \ | /
              \|/
