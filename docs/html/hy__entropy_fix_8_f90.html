NAME

hy_entropyFix

DESCRIPTION

The entropy fix is required for an approximate Roe-type Riemann solver to keep waves away from violating the entropy condition. When the flow is expanding, this will keep the magnetosonic wave speeds away from zero.

Two different entropy fix methods by (i) Harten and (ii) Harten & Hymann (default) are provided. Users can also choose different method by setting, for example, 

 entropyFixMethod = "HARTEN"

in flash.par.
