NAME

hy_getElecticFields

DESCRIPTION

This routine calculates electric fields based on the high-order Godunov fluxes. A choice of higher-order (formally third order) interpolation scheme to construct electric fields is also available by using Taylor expansion. In flash.par file, the runtime parameter "E_modification" should be declared .TRUE. in order to use this modified electric field construction scheme.

Note that this routine is only required for multidimensional problems, and not needed for 1D simulations.
