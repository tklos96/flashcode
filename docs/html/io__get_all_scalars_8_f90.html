numnumber of scalars to get
namesnames of scalar
valuesscalar values
changedreturned array indicating if values have changed since restart

NAME io_getAllScalars

DESCRIPTION

This function gets all the scalars of a given type and returns an array 'changed' indicating if they have changed since most recent restart. Uses linked list implemented under the hood.
