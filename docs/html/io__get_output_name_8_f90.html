NAME io_getOutputName

DESCRIPTION

gets the name of the output file for a checkpoint or plotfile it takes an integer filenum along with 2 strings describing the file, ie 'hdf5' and 'chk' or 'ncmpi' and 'part' and combines it with the number in a string format

EXAMPLE
