NAME gr_mpoleDumpMoments

DESCRIPTION

Utility routine to output the regular Moment_R array and the irregular Moment_I array to a text file. The information is written out to a file named basenm_momentDump.txt, where basenm is the runtime parameter for output file names. The file is appended at each time for each iteration.

NOTES

At the end of each iteration time, the line — finished present iteration — is inserted to make post-processing easier. The original distinctive phrase "Chakka Khan Chakka Khan I feel for you this is the end of this timestep" has NOT been retained. Sorry, but I am open for discussion about including it again.
