Detailed Description

NAME block_1lev_iterator

DESCRIPTION A class that defines an iterator facade around the AMReX MFIter (amrex_mfiter) such that client code may use the iterator for sequentially accessing specific blocks or tiles in the domain that exist only at a single, given refinement level.

Note that this iterator is meant only for internal FLASH use with block_iterator_t. No other code should need to use this code directly.
