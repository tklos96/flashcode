Detailed Description

use gr_flashHook_interfaces

NAME

gr_flashHook_interfaces

DESCRIPTION

Interfaces for some hooks that are called back from within PARAMESH 4, so that FLASH can provide the conversion to (and from) conserved form in mesh interpolation.

NOTES

Names and organization of these subprograms could change, if calling them becomes part of official PARAMESH code. Currently the calls to these routines are FLASH modifications of the PARAMESH code.
