Detailed Description

use gr_physicalMultifabs

NAME gr_physicalMultifabs

DESCRIPTION

Define variables that store physical data as arrays of AMReX multifabs or as arrays of AMReX flux registers.

In all cases, the array index is the refinement level of the associated element. Following AMReX, the level zero corresponds to the coarsest level of refinement.
