Detailed Description

use gr_ptMapData

NAME

gr_ptMapData

DESCRIPTION

Data module for the variables used to map particles to the grid.

NOTES

When compiling on an IBM platform let the compiler perform the appropriate alignment. The same is true for any RISC processor. Do not add the "sequence" keyword to the data structure as this will override any requested compiler alignment. We require natural data alignment, i.e 8 byte values on 8 byte boundary.
