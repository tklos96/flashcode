Detailed Description

nameValueLL_data is a fortran module In addition to holding runtime data this module holds a few overloaded functions as well. In order for overloading to work correctly and portably in fortran the functions need to be inside of a fortran module

NAME nameValueLL_data

DESCRIPTION

Utility module that creates a linked list to hold runtime parameters and scalars

Arguments none
