Detailed Description

use physicaldata

NAME physicaldata

DESCRIPTION

This is the data modules that defines data structures for the physical variables on the discretized mesh for Uniform Grid. The cell centered data structure is "unk", the three face centered ones are "facevarx", "facevary" and "facevarz". There is also a data strucure to provide global domain scope scratch space called "scratch"
