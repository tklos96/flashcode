Detailed Description

use ut_testDriverMod

NAME

ut_testDriverMod

DESCRIPTION

A module that encapsulates common routines and variables that can be used in unittests. Typically, this module is used in the unittests version of Driver_evolveFlash as follows 

<declare variables and setup for test>
...
call start_test_run
...
call assertFalse(didAbort, "Action aborted")
call assertAlmostEqual(x, 1.1, 1.0d-15, "Incorrect x value")
...
call finish_test_run
...
<clean-up>

NOTES

The finish_test_run prints results to standard output.
