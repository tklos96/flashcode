NAME

io_ptWriteParticleData

DESCRIPTION

This routine writes out the particle data in a separate hdf5 file It calls ioh5_write_particles This is being done to make particles easy to debug and since the particles are not associated with the mesh data it makes since to separate it from the checkpoint files and plotfiles

NOTES To control particle output there are a few different parameters
