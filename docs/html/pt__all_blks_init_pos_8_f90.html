NAME pt_allBlksInitPos

DESCRIPTION

Initialize particle positions for all particles.

This default implementation just loops over all local LEAF blocks and does a separate particle initialization by calling Particles_initPositions for each of them.

pt_maxPerProc  INTEGER [100]   Maximum number of particles per processor

NOTES

The particles array should be allocated and initialized to NONEXISTENT before entry. pt_numLocal should have been set to 0. It will be set to the actual number of particles initialized on return.
