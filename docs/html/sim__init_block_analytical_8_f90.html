NAME

sim_initBlockAnalytical

DESCRIPTION

Provides analytical gravity potential solution per block for the Maclaurin spheroid problem. Analytical solution given in: Ricker, P. 2006 "A Direct Multigrid Poisson Solver for Oct-tree adaptive meshes".

References: Maclaurin, C. 1742, Chandrasekhar, S. 1987, Ellipsoidal Figures of Equilibrium

NOTES

The following geometries can be handled: 

               3D cartesian
               3D cylindrical
               2D cylindrical
               2D spherical
