NAME

sim_ranmar

DESCRIPTION

marsaglia and zaman random number generator. period is 2**43 with 900 million different sequences the state of the generator (for restarts) is in the common block.

NOTES ijkl = 54217137 is the same as the four seeds i=12, j=34, k=56, l=78 of marsagalia and zaman

NOTES See paper: Timmes, FX; Zingale, M; Olson, K; Fryxell, B; The Astrophysical Journal, Nov. 10, 2000 : 543: 938-954
