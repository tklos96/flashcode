NAME

sim_setAnalyticSedov

DESCRIPTION Given a set of arrays to store radius, density, pressure, and velocity, together with a time and a ratio of specific heats, generate the analytical solution to the Sedov problem.

Currently this routine is limited to gamma=1.4 (there is hardwired dimensionless constant beta) and will complain if it gets anything else.
