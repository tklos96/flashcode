NAME

sim_x2PolynomialsRootsTest

DESCRIPTION

This routine tests the x2 polynomial root solver on a bunch of difficult to solve x2 polynomials (widespread coefficients, near degenerate roots, etc...). The results will be printed to a special printout file. The obtained x2 polynomial roots will be compared to the exact analytical roots.
