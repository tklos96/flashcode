valuethe number of places a timer can appear

NAME tmr_getMaxTimerParents

DESCRIPTION Accessor function for how many places a timer can appear in various call stacks. For instance, timers A, B, and C appear in a run, as follows:

/A /B/A /C/B/A /B/C

then A has three parents, B has two, and C has two. This number is in general statically limited.
