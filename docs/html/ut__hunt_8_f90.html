call ut_hunt(xx, n, x, low) FUNCTION Given an array xx of length n and a value x, this routine returns a value low such that x is between xx(low) and xx(low+1). the array xx must be

monotonic. low=0 or low=n indicates that x is out of range. on input low is a

guess for the table entry, and should be used as input for the next hunt.

INPUTS xx - real, sorted, array of dimension n to search through n - size of xx x - value to search for low - largest value in [1,n] s.t. xx[low] < x RESULT most errors return low = 1

HISTORY
