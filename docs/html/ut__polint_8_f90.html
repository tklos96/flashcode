NAME ut_polint

DESCRIPTION Polynomial interpolation routine with the same interface as POLINT from Numerical Recipies.

Only linear and quadratic interpolation are actually implemented.

FUNCTION Given n-element arrays (xa,ya) of locations and values, and an interpolation point x, returns a polynomial interpolant (or extrapolant – no range checking done!) of the function in y.
